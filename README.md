# cam-standalone-terraform

##*Currently WIP*

Stacks for deploying CAM Standalone:
- AWS
- Azure
- GCP

Builds basic network infra and server. 

Future: will add DC server to connect.

## Requirements
### Tooling
- Terraform >= 0.12.26

## Usage
Initialize the environment

```
terraform init
```

Planning

```
terraform plan
```

Apply

```
terraform apply
```
